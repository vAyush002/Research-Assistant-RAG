# 🤖 Smart Research Assistant

A Retrieval-Augmented Generation (RAG) app that answers questions about **your own
documents**. Upload PDFs or text files, and ask questions — answers are generated by
the LLM provider you choose (**Groq**, **Anthropic Claude**, or **OpenAI**) using only
the relevant passages retrieved from your documents.

It runs as a **single Streamlit app** — no separate backend to start.

## ✨ Features

- **Multi-provider** — pick Groq / Claude / OpenAI in the sidebar and paste that
  provider's API key (kept in session memory only, never written to disk).
- **Local embeddings** — retrieval uses `sentence-transformers` (all-MiniLM-L6-v2)
  + ChromaDB locally, so an API key is needed **only to generate answers**, not to
  upload or search documents.
- **Sources & relevance** — every answer shows which documents it came from and their
  similarity scores.
- **Optional REST API** — a FastAPI backend (`main.py`) is included for programmatic use.

## 🚀 Quick start

```bash
# 1. Create and activate a virtual environment
python -m venv venv
venv\Scripts\activate          # Windows
# source venv/bin/activate     # macOS / Linux

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
streamlit run streamlit_app.py
```

Then in the browser:
1. In the sidebar, choose a provider, paste your API key, and pick a model.
2. Upload one or more PDF/TXT files.
3. Ask a question and click **Get Answer**.

> On Windows you can also just double-click **`run_app.bat`**.

### Getting an API key

| Provider | Where |
|----------|-------|
| Groq (free tier) | https://console.groq.com/keys |
| Anthropic (Claude) | https://console.anthropic.com/settings/keys |
| OpenAI | https://platform.openai.com/api-keys |

## 📁 Project structure

```
streamlit_app.py        # The app (run this)
rag_pipeline.py         # RAG pipeline + build_llm() provider factory
embeddings.py           # Local sentence-transformers embeddings
vector_store.py         # In-memory NumPy cosine-similarity vector store
preprocessing.py        # Text cleaning + chunking
utils.py                # Config / env helpers (for the optional backend)
main.py                 # Optional FastAPI REST backend
load_sample_documents.py# Bulk-load a folder of docs into the knowledge base
requirements.txt        # Pinned dependencies
.env.example            # Env template (only needed for the FastAPI backend)
DEPLOY_AWS.md           # Deployment notes
```

## 🔌 Optional: FastAPI backend

The Streamlit app is self-contained and does **not** need this. If you want a REST API:

```bash
cp .env.example .env      # then set GROQ_API_KEY
uvicorn main:app --reload # http://localhost:8000  (docs at /docs)
```

## 🔒 Security

- Enter API keys **only in the sidebar** — they live in session memory and are never
  written to disk or committed.
- `.env` is git-ignored. Never commit real keys. If a key is ever exposed, **rotate it**
  at the provider console immediately.
